# AI Employees Plugins

Hermes: this directory contains AI employee plugins for cross-border e-commerce — built on the knowledge-work-plugins architecture, compatible with both Hermes and Codex.

## Quick Install
```bash
bash install-hermes.sh
```

## Available Skills
- `/skill ticket-triage` — Triage support tickets
- `/skill draft-response` — Draft customer responses
- `/skill creator-research` — Research influencer/KOL profiles
- `/skill draft-first-touch` — Draft influencer outreach
- `/skill analyze-video` — Analyze ad creative videos
- `/skill seo-audit` — Shopify SEO audit
- `/skill qualify-lead` — Qualify B2B leads
- `/skill golden-set` — Run evaluation golden set tests
