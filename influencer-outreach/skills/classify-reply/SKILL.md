# Classify Reply

Classify creator replies and determine next actions.

## Reply Categories

| Category | Signal | Next Action |
|----------|--------|-------------|
| **Interested — Yes** | "Love it", address provided | Ship product → 14-day follow-up |
| **Interested — Soft** | "Tell me more", asks questions | Answer questions → keep warm |
| **Negotiating** | "What's the rate?", "My fee is" | Escalate to human |
| **Not Interested** | "Not right now", "Pass" | Archive, no re-pitch for 90 days |
| **Question — Product** | "What settings?", "Results?" | Answer with product facts |
| **Ghost** | No reply after 7 days | Gentle follow-up (1x only) |
| **Already Partnered** | "I work with [competitor]" | Remove from list |
